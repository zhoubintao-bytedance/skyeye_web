# Lessons

- 2026-04-06：首页品牌主句要像使命句，短小、直接、可记忆，避免抽象方法论和明显 AI 腔；本轮确定的方向是“让每一次投资，都更有把握”。
- 2026-04-06：首页产品图不能只是装饰，必须和正式文本一一对齐；至少同时展示红利低波与 TX1 两条产品线的真实图表或运行截图，并支持点击放大预览。
- 2026-04-06：首页中段 section 的背景节奏优先连续黑底，白色更多作为卡片或图片容器的对比色，不再机械沿用黑白交替。
- 2026-04-06：首页信任层由真人团队和公司资质承担主叙事；四位 Agent 可以展示头像，但只能作为辅助能力层出现，不能单独做成喧宾夺主的主 section。
- 2026-04-06：四位 Agent 的协同说明应挂在“团队与背书”语境下，作为真人团队之后的辅助协同层出现；头像必须压小，避免大圆头像抢视觉。
- 2026-04-06：团队区如果是左右两列，上方内容可以分列，但底部的辅助协同面板应横跨整行，不能只挂在右列下方，否则会形成明显残缺感。
- 2026-04-06：四位 Agent 在桌面端应优先单行四列展示；两行布局会显得零碎且不够利落。
- 2026-04-06：产品证据区允许按产品差异展示不同数量的图片，不要机械要求每张卡都必须有上下两张图。
- 2026-04-06：首页导航中 `research-notes` 的中文标签统一使用“博客”，不再使用“最新研究”。
- 2026-04-06：首页里所有对外可见的“联系合作”统一改成“加入我们”，包括导航、CTA 和联系区标题。
- 2026-04-06：Agent Team 卡片里的头像不能太小，也不能贴在左上角；桌面端应优先放在卡片中轴位置，并与文字形成居中关系。
- 2026-04-06：如果 Agent Team 与创始人区同时出现，优先复用创始人卡片的排版骨架，保持头像、字体和信息层级一致，不要单独发明另一套卡片语言。
- 2026-04-06：Agent Team 里上方绿色标签统一使用四字角色名；Agent 名字不要做大标题，应该放进下方介绍句里，整体比创始人卡片更克制、更优雅。
- 2026-04-06：Agent Team 模块头部不需要额外的大标题和说明句，保留一个绿色小字 `Agent员工协同` 即可，整体更干净。
- 2026-04-06：产品卡右侧配图替换时，左侧摘要、亮点和图标题也要同步跟随图片语义更新，避免图是回测、字还在写收益曲线。
- 2026-04-06：首页主图替换时，首屏图标题、细说明和预览标题都要同步更新，避免图已经是回测总览但文字仍写旧的 performance 口径。
- 2026-04-06：首页主图替换后，首屏四个摘要指标也必须同步跟图上的回测数值更新；其中累计收益和年化收益要用绿色高亮。
- 2026-04-06：首页不应同时承载博客文章列表和 `/blog` 独立文章页；如果 `/blog` 已是完整承载页，首页应删除重复文章区，只保留一个清晰的 `/blog` 入口。
- 2026-04-06：站点头部品牌 logo 不能偏小；在黑底导航上应至少达到清晰可见的主品牌体量，优先保证 logo 和公司名形成稳定的并列识别。
- 2026-04-07：当用户要求“文字微调”时，优先做最小改动，不要大范围重写整页语气；除非用户明确要换风格，否则保留原有表达，只修格式、错别字和明显不通顺处。
- 2026-04-07：团队区的对齐不能靠把资质图片卡强行拉高；资质类图片应保持自然高度，通过统一间距和卡片内边距解决节奏问题。
- 2026-04-07：团队区若需要让左侧公司信息和右侧创始人卡上下边线同时对齐，应改成桌面端共享两行的外层网格，而不是让左列整体对齐右列总高度。
- 2026-04-07：创始人职位不要使用 `CEO / 首席执行官` 这种会被浏览器随意拆行的整串文本；应拆成英文缩写徽标和中文全称两层展示。
- 2026-04-07：创始人姓名也要强制单行显示；中文姓名在窄列里会被浏览器逐字换行，必须显式加 `whitespace-nowrap`。
- 2026-04-07：博客列表页如果同时承载多篇文章，不要用 `featured + secondary` 的横向不等宽排布；文章卡应采用同一模板、同一宽度，按垂直单列依次排列。
- 2026-04-07：博客列表页文章顺序必须按发布日期倒序，最新文章在最上面，旧文章在下面。
- 2026-04-07：博客详情页的头部摘要区域要走统一模板，不要只给个别文章额外叠一张很长的“核心摘要”侧卡。
- 2026-04-07：诗歌类文章在详情页头部不要直接展示会剧透正文气质的摘要句，避免列表外预览过度透出内容。
- 2026-04-07：博客详情页返回操作不要用普通文本箭头链接，应改成更贴近 NVIDIA 风格的绿色描边按钮。
- 2026-04-07：博客列表里的诗歌文章摘要也必须避免剧透正文句子，列表卡片应只展示短摘要，不要直接露出“诗歌正文感”的长句。
- 2026-04-07：博客列表页标题不要同时出现一小一大两个“博客”；应合并成一个主标题，并允许用品牌绿色背景做醒目标识，同时删除多余说明句。
